# Development Tasks
## 소설 쓰기 웹 애플리케이션 - MVP

---

## 1. 프로젝트 개요

### 1.1 목표

**2-3개월 내 MVP 출시**를 목표로 핵심 기능만 구현합니다.

### 1.2 핵심 기능 (MVP)

1. ✅ 사용자 인증 (이메일 + 소셜 로그인)
2. ✅ 프로젝트 CRUD
3. ✅ 챕터 CRUD
4. ✅ 리치 텍스트 에디터 (자동 저장)
5. ✅ 단어 수 카운팅
6. ✅ TXT/DOCX 내보내기
7. ✅ 반응형 디자인
8. ✅ 다크 모드
